## Introduction

Based on TrinityCore, a *MMORPG* Framework. ISC aims to adapt the framework for a SinglePlayer experience by reusing and modifying the existing codebase.

This is primarily **an educational project**. No commercial use or massive distribution is supported or encouraged by this project.
